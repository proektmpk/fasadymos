# Рельефная Москва — сайт исследовательского проекта

Сайт создан на основе индивидуального проекта студентки ГБПОУ «Московский педагогический колледж» Полуектовой Алёны Евгеньевны.

**Тема:** Московские фасады начала XX века: роль рельефного декора в формировании городского пространства.

**Стек:** Статический HTML/CSS, GitHub Pages.

## Структура сайта
- `index.html` — главная страница
- `about.html` — о проекте
- `styles.html` — стили и смыслы (модерн, неоклассицизм, конструктивизм)
- `map.html` — маршрут «Зелёный коридор»
- `condition.html` — состояние и сохранение декора
- `survey.html` — опрос и результаты
- `teachers.html` — материалы для педагогов
- `style.css` — единый файл стилей
- `docs/` — PDF-файлы (полный текст проекта, карта маршрута, анкета, методичка)

## Как запустить локально
Открыть `index.html` в браузере.

## Как опубликовать на GitHub Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main`, folder: `/ (root)`
4. Сохранить. Сайт будет доступен по адресу `https://ваш-логин.github.io/relief-moscow/`
